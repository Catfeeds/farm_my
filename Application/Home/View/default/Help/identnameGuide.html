实名认证修改指南
根据相关法规，在农畜商品交易平台注册的用户必须进行身份认证并保证信息的真实合法性，认证成功后原则上不允许更改，若有特殊情况需要更改请联系星币官方客服。具体步骤如下：

  1. 打开农畜商品交易平台，登录已有账号，点击【安全中心】。

  2. 请用户本人联系客服，客服人员会帮助用户进行取消实名认证操作。

  3. 取消实名认证后，用户须重新登录农畜商品交易平台账户，并在【安全中心】-【实名认证】页面重新实名，如图：

    注: 提现时银行卡信息须与实名认证的身份信息一致，请填写真实身份信息，以免带来不便。

  4. 认证成功后，可在当前页面查看本人的身份信息。
