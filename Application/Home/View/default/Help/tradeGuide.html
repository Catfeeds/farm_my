交易指南
首先，进入华联全球商贸平台首页，登录已有账号，点击进入【交易中心】页面，该页面将显示华联全球商贸平台上所有类别的币种，选择您想交易的币种，如图：

买入指南

  1. 页面跳转到狗狗币交易页面，在买入栏中填入【买入价格】【买入数量】【交易密码】，点击【买入】。系统将自动成交挂单。

    注：买入数量应小于等于【最大可买】。
    虚拟币买入数量不能小于0.1；成交金额不能少于10元。
    不同虚拟币的买入手续费不同，请买入前仔细确认金额。

  2. 挂单成功后，可在【财务中心】-【委托管理】中查询挂单委托情况。

    注：若挂单价格与当前市价不吻合，可能导致挂单不能及时成交。可稍作等待，观察行情。若挂单仍未成交，可撤销（交易中心-我的委托挂单-撤销）挂单调整价格重新挂单。撤销后，未成交部分资产（如买币时冻结的资金、卖币时冻结的币）将原数返还到您的平台账户中

  3. 也可在【财务中心】-【成交查询】中查询挂单成交情况。

  4. 挂单成交后，请刷新页面，点击【财务中心】查看资产变化。

    如有挂单其他问题，可咨询在线客服400-657-8880。

    注：卖出数量应小于等于【最大可买】。 虚拟币买入数量不能小于0.1；成交金额不能少于10元。 不同虚拟币的买入手续费不同，请买入前仔细确认金额。
