登录密码重置指南

找回登录密码

  1. 打开农畜商品交易平台，在登录框右下方，点击【忘记密码】。

  2. 输入注册邮箱账号及验证码，点击【找回密码】。

  3. 当弹出提示【密码重置邮件已发出】时，请登录注册邮箱查看“农畜商品交易平台密码重置”邮件，并点击邮件中【重置您的密码】。

  4. 在找回密码页面输入新登录密码，点击【修改密码】。

  5. 注册成功。登录账户后即可直接进行充值并交易。

修改登录密码

  1. 打开农畜商品交易平台首页，登录已有账号，点击【安全中心】。

  2. 在页面左侧菜单选择【修改交易密码】，按页面提示输入旧登录密码+新登录密码，点击【保存】即可。

    注：建议使用大小写和数字组合的6位以上密码。

  3. 修改成功。请重新登录农畜商品交易平台。
