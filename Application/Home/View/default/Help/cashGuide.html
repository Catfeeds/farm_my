提现指南

  1. 进入农畜商品交易平台首页，登录已有账号，点击【去财务中心】或在导航栏选择【财务中心】，如图：

  2. 人民币提现操作需绑定用户本人的银行卡，在页面左侧菜单选择【人民币提现】，点击【管理银行卡】。若已绑定可直接申请提现。

  3. 进入银行卡管理页面，填写用户本人的银行卡信息（储蓄卡/借记卡）后，点击【添加银行卡】。

    注：姓名栏，系统默认为实名认证姓名。
    若此处的姓名与本人真实姓名不符，请联系在线客服400-657-8880。

  4. 挂银行卡添加成功后，返回【人民币提现】页面，填写提现申请。

    注：提现金额单笔最低可申请100元，最高50000元。提现手续费率为提现金额的0.5％，每笔提现最低收费2元。

  5. 申请提现后，财务会在24小时之内处理提现。请耐心等待并随时关注银行账户资金变动。若超过24小时仍未到账，请联系在线客服400-657-8880。
