充值指南
首先，进入华联全球商贸平台首页，登录已有账号，将鼠标放在【交易中心】按钮上，页面将显示华联全球商贸平台上所有类别的种类，选择您想交易的币种，如图：

人民币充值有两种： 在线自动充值和银行转账充值。以下分别演示两种充值方式流程。

在线自动充值

  1. 在页面左侧菜单选择【人民币在线自动充值】，填写充值金额，点击【到网上银行支付】。

    注：每次充值最少不少于100元，最多不超过5万元。

  2. .确认充值信息，核实无误后点击【确认提交】。

    注：Chrome浏览器可能不支持部分银行的网银控件，建议使用IE内核浏览器进行充值。

  3. 请确认充值金额，并选择银行进行付款。（以招商银行为例）

  4. 根据银行提示完成网上支付。

    • 选择卡号密码支付，输入银行卡号及网上支付密码，完成支付。

      注：部分银行可能需要下载插件，安装插件后，请重新启动浏览器。

    • 选择手机支付，输入手机号码、卡号后四位，完成支付。

      注：提示充值成功后，请刷新华联全球商贸平台页面或重新登录华联全球商贸平台账户查看资产。
      具体到账时间根据不同银行的受理时间而定，请耐心等待。若转账后超过1小时仍未入账，请联系在线客服400-657-8880。

银行转账充值

  1. 在页面左侧菜单选择【人民币银行卡转账充值】，填写充值金额。

    注：汇款人姓名默认为实名认证的姓名，仅允许使用本人的银行卡或支付宝进行转账充值。

  2. 页面弹出平台的收款账号及汇款信息，请根据“银行转账汇款”信息通过网银、去银行柜台或使用支付宝进行转账汇款。
